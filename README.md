# dw-sample-reactor
Template for devops-ready applications.
Default technology setup for: 
* webserver (Jetty)
* IoC, 
* REST (jersey)
* Logging
* Activity Reporting
* Healthchecks
* Monitoring
* Packaging
* Configuration (Constrett)
* ... and more to come.


Extension of the setup found on <a href="http://www.dropwizard.io/"/>DropWiard</a>
